# 2:1 Multiplexer (MUX)



## 📌 Description



A simple 2-to-1 multiplexer that selects one of two inputs based on a control signal.



## 🏗️ Design Overview



* Type: Combinational Logic

* Inputs: `a`, `b`, `s0`

* Output: `y`



## ⚙️ RTL Implementation



* Implemented using conditional operator

* Output switches between inputs based on select signal



## 🧪 Verification Strategy



* Randomized selection and data inputs

* Scoreboard validates correct data routing



## 📊 Results



* Verified correct selection behavior under all conditions
